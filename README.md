# 100 Days of DSA 🚀

Welcome to my **100 Days of DSA** challenge repository! 

This repository was created as part of my **College Internal Assessment**. The challenge is to solve **2 Data Structures and Algorithms (DSA) questions every single day for 100 consecutive days**. 

The problem set is a mix of questions curated from popular coding platforms like **LeetCode** and **GeeksforGeeks (GFG)**, ranging from basic array manipulations to complex algorithmic challenges.

## 🎯 Challenge Goals
- Solve exactly **2 DSA problems** every day.
- Maintain consistency for **100 days**.
- Write clean, efficient, and well-documented C code.
- Improve problem-solving skills and master core data structures.

## 💻 Tech Stack
- **Language:** C programming

## 📁 Repository Structure
The repository is structured day by day. Each day's folder contains the source code for the problems solved on that specific day.

```text

```
